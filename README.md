# cypherix.info
A simple website for showcasing events, teams, and information about Cypherix.

## Overview
This project serves as a basic website to display information about events, teams, and general details about Cypherix.

## Features
- Display upcoming events
- Showcase different teams
- Provide information about Cypherix
